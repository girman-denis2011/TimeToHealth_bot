Health Reminder Telegram Bot made with Vibe coding

A customizable Telegram bot that helps users stay healthy by sending reminders to stretch, drink water, walk, and take breaks throughout the day.

Built with python-telegram-bot, this bot supports multi-language, user preferences, and smart scheduling.

Features
Time-based reminders (default)
Inactivity-based mode (configurable)
Multiple reminder types:
Stretch
Water
Walk
Break
Multi-language support
English 🇺🇸
Russian 🇷🇺
⚙️ User settings menu (inline buttons)
🧠 Motivational quotes included
📊 Persistent user settings (JSON storage)
🔁 Adjustable reminder frequency
🕒 Active hours control
⏯️ Start/stop bot anytime
